# IBM Bob - Getting Started Guide

## Overview

This guide is the starting point for learning IBM Bob. It is designed as a simple learning path for new users who want to install Bob, start using it in a real repository, add project instructions, and then build reusable skills over time.

## Objective

Use this page as an outline:

1. Install Bob and Bobshell, then open your first repository in the Bob IDE.
2. Add an `AGENTS.md` file so Bob understands your project's rules and workflow.
3. Learn how Bob skills work and how to create reusable templates for common tasks.

## Learning Path

### Guide #1: Install Bob, Bobshell, and Open Your First Repository

Start here if you have not installed Bob yet. This guide walks through downloading Bob, installing bobshell, launching the CLI, cloning a repository, and opening that repository in the Bob IDE.

- Guide file: [IBM Bob Guide #1 - Install Bob, Bobshell, and Open Your First Repository](./01-install-ibm-bob-guide.md)
- Covers Bob installation, bobshell setup, first-run validation, Bob tokens, and usage analytics
- Includes references for both IDE and shell workflows so users can keep learning after the initial setup

### Guide #2: Create an `AGENTS.md` File and Project Instructions

Once Bob is installed, the next step is teaching Bob how your project works. This guide explains what `AGENTS.md` is, why it matters, how to write one, and how `/init` can generate more detailed instruction files for different modes.

- Guide file: [IBM Bob Guide #2 - Create AGENTS.md and Project Rules](./02-create-agents-guide.md)
- Covers a starter `AGENTS.md` example, recommended sections, and how project-specific instructions improve results
- Includes references for `/init` and mode-specific instruction files

### Guide #3: Create and Use Skills with Templates

After Bob understands your project rules, the next step is teaching it reusable workflows. This guide explains how Bob skills are structured, where they live in a repository, and how to create simple `SKILL.md` templates for repeatable tasks.

- Guide file: [IBM Bob Guide #3 - Skills and Templates Guide](./03-create-skills-guide.md)
- Covers skill layout, minimum file format, sample templates, and ideas for expanding skills over time
- Includes official Bob documentation plus external references for deeper research
